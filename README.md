# CC Rewriter Bot

Inoltra i post dei seller al bot in privato.
Il bot ripubblica nel gruppo "POST PRONTI" con link Mulebuy/CNfans riscritti col tuo ref e max 4 foto.

Variabili ambiente richieste:
- BOT_TOKEN
- OWNER_USER_ID
- OUTPUT_CHAT_ID
- MULEBUY_REF
- CNFANS_REF
- TZ
- MAX_PHOTOS
- MEDIA_GROUP_WAIT
